StatusCode init_master_tasks ( )

Initialize all 3 master tasks.

The three tasks run at 50hz (1ms), 3hz (100ms) and 1hz (1s)

Returns
STATUS_CODE_OK if initialization succeeded for all 3 tasks STATUS_CODE_INVALID_ARGS if initialization failed for atleast 1 of the 3 tasks
